=======
Clients
=======

This directory holds the source code and project files for platform specific
Plexnet clients.

Whilst only a few are currently being actively developed, the plan is to support
all popular platforms including:

* Android
* iPhone
* Facebook
* Firefox
* FreeBSD
* Linux
* OS X
* Pre
* Symbian (including Series 60)
* Unix (other variants)
* Windows
